Phenotyping Analysis: Pearl Millet Panicles

University of Massachusetts Boston | Nov 2021 - Feb 2022

Automated measurement extraction from Pearl millet panicle images using PlantCV, increasing monitoring efficiency by 80%.

Project Overview

By leveraging the PlantCV library, I automated the analysis of Pearl millet panicle images, extracting key measurements for agricultural research. This project demonstrated how automation could enhance food security through better monitoring of millet breeding programs.

Presented results at the Sixth CSM Student Success Showcase.
